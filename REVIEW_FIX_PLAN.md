# Transcription Review Fix Plan

Date: 2026-03-14

## Goal

文字起こしロジックのレビューで見つかった問題を、起動不能や誤出力に直結するものから順に修正する。

## Issues

1. Whisper未インストール時に `WhisperService()` の初期化でアプリ起動が失敗する。
2. 日本語テキストのオーバーラップ検出が単語分割前提のため機能しにくい。
3. Whisper分割処理でエラープレースホルダーがそのまま出力本文に混入する。
4. セグメント処理で一部失敗しても成功扱いになり、全失敗時は空出力になり得る。
5. Gemini使用量記録のモデル名が固定で、実際の選択モデルを反映しない。
6. Gemini分割処理で安全フィルターや著作権ブロックが一般エラーとして握りつぶされる。
7. タイトル生成時の `genai.configure()` がグローバル設定に依存して競合リスクがある。
8. セグメントキャッシュを再利用した後に cleanup で削除してしまい、キャッシュの価値が落ちる。

## Fix Order

1. [done] Whisper初期化を遅延化して起動不能を解消する。
2. [done] セグメントエラー処理を整理し、Whisper/Geminiともに部分失敗を成功扱いしない。
3. [done] 日本語向けのオーバーラップ検出を改善する。
4. [done] モデル名追跡とタイトル生成の設定競合を直す。
5. [done] セグメントキャッシュの cleanup 条件を見直す。
6. [done] コンパイルとターゲット検証で回帰確認する。
